# Alexandre Gonçalves - Personal Website

A minimalist, static personal website containing a brief CV/timeline, links to pet projects/blog, and social profiles.

## Technologies Used
- HTML5
- CSS3 (Vanilla, No frameworks)

## Deployment (Cloudflare Pages)
This site is designed to be easily deployed to Cloudflare Pages:
1. Push this repository to GitHub.
2. Log into Cloudflare Pages and connect the GitHub repository.
3. Configure the build settings (none required for static HTML/CSS).
4. Deploy and attach a custom domain if desired.
